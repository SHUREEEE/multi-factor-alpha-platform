# Contributing

This is a personal research and portfolio project. Issues, questions, and
review notes are welcome; pull requests are reviewed case by case.

Please keep contributions aligned with the project's research boundary:

- Do not add unsupported alpha, attribution, or live-readiness claims.
- Keep gross/net performance, data limitations, and launch blockers explicit.
- Do not commit large generated datasets, binary research panels, or local
  cache outputs.
- Include focused tests or a reproducible command when changing behavior.
- Update docs when changing reviewer-facing results, workflows, or claims.
